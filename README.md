A mix of creative projects—from animation reels and branded content to cross-platform mobile apps using React Native and Expo. I’m also experimenting with 3D motion graphics in Blender and Cinema 4D.  

👯 I’m looking to collaborate on  
Creative branding projects, UI/UX-focused mobile apps, animation-driven storytelling, and indie game development using Unity 3D.  

🤝 I’m looking for help with  
Expanding my reach in the creative tech space—especially with advanced 3D animation workflows and scalable mobile app architecture.

Advanced C# for Unity, procedural animation in Blender, and performance optimization in React Native for large-scale apps.

Logo animation, branding design, building with React Native, editing in After Effects, or blending 2D/3D workflows for visual storytelling.
